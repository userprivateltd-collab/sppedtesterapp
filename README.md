# KnowIt - Network Matrix

A premium, clean, and high-performance network speed testing application built with Jetpack Compose.

## 🚀 Features
- **Real-time Telemetry:** Smooth, physics-based gauge animations.
- **Modern UI:** A sleek "Premium Clean" aesthetic using a custom color palette.
- **Detailed Analytics:** Provides clear Mbps readouts for network performance.
- **Built with Compose:** 100% native Android UI.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Design:** Material 3
- **Animations:** Compose Animation API (animateFloatAsState)
